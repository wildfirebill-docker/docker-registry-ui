Bulk Operations

DANGER: Expert Activity Only

WARNING: Bulk operations can permanently delete images that may still be in use. This is a static rule-based system with NO usage history tracking.

Critical Considerations:

  • Images older than 2 years might be base images for other containers still in production
  • Deleted tags cannot be recovered - they must be rebuilt and re-pushed
  • This tool has no knowledge of which images are actively used in deployments
  • Always use Dry Run mode first to preview what will be deleted
  • "Keep minimum tags" is MANDATORY when using age-based deletion to prevent losing all images
  • Garbage collection must be run manually after deletion - deleted tags are NOT automatically cleaned
Note: Deleting tags only removes manifest references. To free disk space, you must manually run garbage collection on your registry. See Cleanup Instructions for commands.
